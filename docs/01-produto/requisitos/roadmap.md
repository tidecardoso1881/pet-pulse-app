# 🗺️ Roadmap — PetPulse

## MVP (v0.1) — Lançamento inicial

### Autenticação
- [ ] Cadastro de usuário (email + senha)
- [ ] Login / Logout
- [ ] Recuperação de senha
- [ ] Perfil do usuário

### Gestão de Pets
- [ ] Cadastrar pet (nome, espécie, raça, sexo, data de nascimento, foto)
- [ ] Listar pets do usuário
- [ ] Editar dados do pet
- [ ] Excluir pet

### Agendamentos
- [ ] Criar agendamento de consulta
- [ ] Listar agendamentos (próximos e histórico)
- [ ] Editar agendamento
- [ ] Cancelar agendamento

### Vacinas
- [ ] Registrar vacina aplicada
- [ ] Listar histórico de vacinas por pet
- [ ] Alerta de próxima dose

### Veterinários
- [ ] Cadastrar veterinário / clínica
- [ ] Listar veterinários
- [ ] Vincular vet a uma consulta

---

## v1.0 — Pós-MVP

- [ ] Notificações por email (lembrete de consulta, vacina)
- [ ] Upload de exames e laudos (PDF/imagem)
- [ ] Dashboard com resumo de saúde dos pets
- [ ] Compartilhamento de histórico com veterinário
- [ ] App mobile (PWA)
