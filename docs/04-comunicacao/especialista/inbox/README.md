# 📥 Inbox do Especialista (Claude Code)

> **Leia esta pasta SEMPRE ao iniciar uma sessão — antes de qualquer outra coisa.**

## Como usar

1. Liste os arquivos desta pasta
2. Leia cada ticket com status diferente de `[✓] DONE`
3. Execute na ordem de prioridade (Alta → Média → Baixa)
4. Ao concluir cada ticket, escreva um relatório em:
   `docs/04-comunicacao/gerente/inbox/RELATORIO-[ID]_[desc]_[data].md`

## Convenção de nomes

```
TICKET-[ID]_[descricao-curta]_[DDMMAAAA].md

Exemplo:
TICKET-001_auth-login-page_15042026.md
TICKET-002_pets-listing-page_16042026.md
```
